# Kappafy -- JSON Explorer & Mock API Generator

**[-> Use Kappafy (live tool)](https://kappafy.com/)**

Kappafy is a free JSON explorer that turns raw JSON into an interactive collapsible tree view. Click any node to see its JSONPath, analyze document statistics, prettify or minify JSON, and generate mock REST API endpoints from your data structure. Everything runs client-side in your browser.

## Features

- Collapsible JSON tree view with color-coded data types
- JSONPath display for any selected node with copy support
- JSON statistics: total keys, nesting depth, array sizes, type distribution
- Mock REST API endpoint generation (GET, POST, PUT, DELETE)
- One-click JSON prettify and minify
- Copy formatted or minified JSON to clipboard
- 100% client-side -- your data never leaves your browser
- Open source -- inspect the code yourself

## Tech Stack

- Vanilla JavaScript (no frameworks, no build step)
- Static HTML hosted on GitHub Pages
- Cloudflare DNS + SSL
- Zero dependencies, zero tracking, zero cookies
